To run the file first compile all the java files in the folder pack and then run the hello.java file or rather the hello class file that would be generated after compilation.

For changes to any initial_values, just make a change in the file initial_values.java

All values in that file are in seconds and so should be entered in the same way.

All screenshots are taken at each hour of the simulation.

The output can be found in the log.txt file and the snapshots in the snapshot.txt file.
